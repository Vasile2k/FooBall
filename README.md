# FooBall
A game w/8 a ball... a foo ball
## Building
Dependencies:
* [LWJGL](https://github.com/LWJGL/lwjgl3)
* [JDBC](https://bitbucket.org/xerial/sqlite-jdbc/src/default/)
* [JOML](https://github.com/JOML-CI/JOML)
